## Description
Sajni Re is a modern and responsive jewellery brand website inspired by Indian craftsmanship and contemporary fashion.  
The project focuses on elegant UI, smooth navigation, and a premium shopping experience.

## Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript

## Features
- Elegant and user-friendly UI
- Fully responsive design
- Product showcase with modern layouts
- Indian-inspired visual aesthetics
- Reusable React components

## Getting Started
To run this project locally:

```bash
npm install
npm run dev
```
